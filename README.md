## Endpoints

##### Start parking meter

`/start?vehicleNumber=KR12345&driverType=REGULAR`

##### Stop parking meter

`/stop?vehicleNumber=KR12345

##### Check if parking meter was started

`/check?vehicleNumber=KR12345`

##### Check income for given day

`/income?date=2018-09-17`
